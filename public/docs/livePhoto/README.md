# LifePalette iphone实况图片上传教程

- 🎈 LifePalette支持实况图片上传，实况图片上传后，在web端和移动端都可以预览，方便用户分享和展示实况图片。

- web端通过Icloud进行相册访问下载，下载的压缩包有同样的MOV与HEIC两个文件，之后步骤同步骤5-7
