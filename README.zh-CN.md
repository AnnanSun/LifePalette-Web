## 🚀 Origin

- 🎈 记录自己的生活点滴

## 🦄 功能

- ⚡ **扫码登录**: 后台使用 nestjs,接入扫码登录
- 🐱 **阿里 oss 文件上传**: 基于`ali-oss`的 [Node.js](https://help.aliyun.com/document_detail/32067.html?spm=a2c4g.32070.0.0.607a55afYXWVU3)实现文件上传
- 🎈 **短信验证**: 通过`阿里大于`实现短信发送以及验证功能
- 🥏 **动态发布**: 发布图文视频
  -🚩 **个人信息修改**: 待开发...
